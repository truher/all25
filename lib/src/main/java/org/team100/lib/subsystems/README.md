# lib.subsystems

General, configurable subsystems you can use to assemble your robot code:

* [five_bar](five_bar/README.md): a 2d mechanism used for control training
* [led](led/DemoLED.java): a very simple LED subsystem for control training
* [lynxmotion_arm](lynxmotion_arm/README.md): a servo arm used for control training
* [mecanum](mecanum/README.md): Mecanum drive used in Mechanical Mayhem
* [prr](prr/README.md): 2-dof arm on an elevator
* [r3](r3/README.md): general interfaces for 3d (holonomic) control
* [shooter](shooter/README.md): examples of shooter implementations
* [swerve](swerve/README.md): the swerve code we use for competition
* [tank](tank/README.md): tank-drive robots, for demo and training
* [turret](turret/Turret.java): a simple auto-aiming turret
