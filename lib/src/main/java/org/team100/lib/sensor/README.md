# lib.sensor

All sensors (except cameras) go here:

* [distance](distance/README.md)
  * distance to some remote object
* [gyro](gyro/README.md)
  * absolute rotation relative to the earth
* [interrupt](interrupt/README.md)
  * a photo-interruptor ("beam break sensor")
* [position](position/README.md)
  * absolute and relative rotation of a joint
* [switch100](switch100/README.md)
  * a simple electrical switch
